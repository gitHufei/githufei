了解js

  • javascript
  • js是运行在浏览器（运行环境）
  • js是唯一一门可以和用户直接交互的语言
  • js是一门轻量级的解释型脚本语言
    • 解释型：代码边解释边运行
    • 轻量级：语言不严谨
  • js安全性低，很容易被篡改
  • js是单线程语言（从上到下执行）

js的组成

  • ECMAScript ，ECMA是一个组成，这个组织规定了js的基本规范 – ECMAScript 实际上就是js的API（js的规范）
  • DOM Document Object Model 文档对象模型 是操作document（文档对象） 文档对象是文件的顶级对象
  • BOM 浏览器对象模型(Browser Object Model) 操作的是浏览器 ，浏览器的顶级对象是window（浏览器对象）

js的引入方式

js是不能独立运行的，需要依靠html文档运行在浏览器中，所以我们需要将js引入到html中

  • 行内式
  • 内嵌式
  • 外联式

js代码一般写在整个html文档的最后面，因为要先加载document的所有内容，然后再将js添加到每一个DOM对象上

如果将js代码写在外联式的元素中间，这部分代码是无效的

js的输出方式

让用户察觉到页面的变化

  • alert 弹出
  • document.write(“输出的内容【html元素】【文字】”)

  • 浏览器控制台输出（console）

    • console.log() 向控制台输出一条信息
    • console.dir() 向控制台输出一个对象的所有内置属性
      • 内置属性：天生自带的属性，就向人有一个鼻子，一对眼睛，内置属性可以改变属性值
      • 自定义属性：人为给这个对象添加的属性名和属性值，例如金*给自己定义了一个新的属性，第三性别：男女究极混合体

  • innerHTML 可以获取指定元素内容和向指定元素内添加内容

  • innerText 可以获取指定元素内容和向指定元素内添加内容

    innerHTML可以获取一个元素内的所有内容（包括元素和文本）
    innerText只能获取一个元素内的文本内容

获取的语法：
元素.innerHTML / 元素.innerText

添加的语法：
元素.innerHTML = 要添加的内容
元素.innerText = 要添加的内容

在js中 等号 是赋值的意思，将等号右侧的值赋给等号左侧
innerHTML、innerText在向元素内添加内容的时候，会将原来的内容替换掉

js的命名规范

在中国，命名的时候可以使用拼音全拼
耿银鹏 gyp（鬼知道这三个字母代表神马） gengyinpeng

  • 严格区分大小写 test、Test代表了不同的对象
  • 不能以数字为开头 1zfpx 错误的 zfpx1 正确的

数字 number

哪些值是number？
整数 0 小数 NaN （not a number）

判断一个值是否为数字：isNaN()
isNaN –>xxx is not a number
返回值
符合条件为true – 不是一个数字
不符合条件为false – 是一个数字

isNaN(abc) – > 此时abc 浏览器认为这是一个变量
abc is not defined abc没有被定义过

在js中，除了变量名之外的所有内容基本上都是字符串，字符串在js中需要用引号包起来

isNaN(‘0’) – > false
isNaN() 在判断之前会将传进来的值先进行转化，将其他数据类型转为数字数据类型
对象 object – > number

  • 将对象转为字符 调用 toString()方法
  • 再将字符串转为数字 调用 Number()方法
    {} –> “[object Object]” – > NaN

    [] –> “” –> 0

字符串 string – > number

  • 字符串转为数字 调用 Number()方法

布尔 boolean – > number 调用 Number()方法

  • true 为 1
  • false 为 0

null – > number 调用 Number()方法

  • null 为 0

undefined – > number 调用 Number()方法

  • undefined 为 NaN

Number()这个方法的返回值
要么是数字 要么是NaN
如果传入的值有一个部分不属于数字，那么整体否定，返回NaN，例如 Number(“1a2b”) Number(“1.3.3”) Number(“100,000.00”) 返回都是NaN

isNaN()的返回值
要么是true 要么是false

parseFloat() 浮点数（小数）
parseInt() 整数

以上两个方法，都是从指定值中去分离一部分数字，一般指定的值要么是数字要么是字符串,如果传入的值不是数字也不是字符串，先将值转为字符串，然后在字符串中取数字

工作机制：将值从第一个查找到最后一个，将有效数字部分返回，Float可以识别小数，Int只识别整数部分

返回值：要么是数字 要么是NaN

Number(value) 将其他数据类型转为数字数据类型
返回值：数字 NaN

parseFloat(value)/parseInt(value) 从第一个值向后查找，直到值中不是数字类型的停止查找，将查找到的部分返回
parseFloat(value) 浮点数
parseInt(value) 整数
返回值：数字 NaN

isNaN(value) 判断一个值不是数字
返回值：
true value不是数字
false value是数字
如果传值不是一个数字数据类型，那么isNaN会先将传进的值转为数字数据类型（调用的是Number方法）

parseInt(“.36”) – > NaN
parseInt(.36) – > 0

布尔 boolean

只有两个值 true（真） false（假）
哪些值为假，哪些值为真
0 NaN “” null undefined false为假，其余都为真

怎样判断一个值是真是假（将其他数据类型转为布尔类型的方法）
Boolean(value)将其他数据类型转为布尔类型的方法

在进行布尔运算的时候，传入的值不需要进行转化

逻辑运算：
!value 先将value转为布尔，然后取反
!0 –> !Boolean(0) –> !false –> true
!!value 先将value转为布尔，然后取反再取反
!!0 –> !!Boolean(0) –> !!false –> !true –> false

!! 的作用和Boolean()作用一致，都是将其他数据类型转为布尔类型

字符串 string

字符串：在js中字符串是一个变量，用来储存文本的，用引号包裹，字符串在js中不具有任何的意义，也没有任何的功能

xxx.toString() 将其他数据类型转为字符串
{} – > “[object Object]”
[] – > “”
0 –> “0”
null – > “null”
undefined – > “undefined”
拼接字符串就是将两个字符串用 “+”链接起来
如果是用变量来代表字符串的时候，变量不能用引号包起来

其他数据类型如果遇到字符串拼接的时候，会默认先转为字符串数据类型，然后在拼接
例如 []+’abc’ – > [].toString()+’abc’ – >’’+”abc” –>”abc”
{}+”0” –> {}.toString()+”0” –> “[object object]” + “0”–>”[object object]0”

null和undefined

很有趣在js中有两个表示“无”的值
null表示”没有对象”，即该处不应该有值（现在预留，之后可以添加）
null也可以进行保留对象并将值清空
var obj=null;
undefined表示”缺少值”，就是此处应该有一个值，但是还没有定义
var obj; –> console.log(obj); –> undefined

Number(null) – > 0
Number(undefined) – > NaN

总结：
number

  • Number(value)
  • parseFloat(value)
  • parseInt(value)
  • isNaN(value)

boolean

  • Boolean()
  • !value
  • !!value
  • 五个值为假（0 NaN ‘’ null undefined） 其余都为真

string

  • 将其他数据类型转为字符串 value.toString()
  • 字符串拼接 用加号 任何数据类型和字符串拼接的时候，都会默认先转为字符串

null / undefined

  • null 现在没有值，以后会有，此时null代表空
    • 0
  • undefined 此处本应有值，但是没有定义这个值
    • NaN

Object 对象类

js中引用数据类型分为对象数据类型和函数数据类型
对象数据类型：

  • 数组类 Array
  • 对象类 Object
  • 数学类 Math
  • 正则类 RegExp

Object 是引用数据类型中的对象类，每一个我们使用的具体对象都是这个类的一个实例，例如object是自然界，Object是人类，石晓丰是人类中的一个具体的实例

对象类中的实例都是由键值对组成的（属性名和属性值），键值对是用来描述这个对象特征的

如何创建一个对象实例：

  • 字面量创建

  • 实例创建 在该类上新建一个实例

  • 访问对象 查找

    • 对象名.属性名
    • 对象名[“属性名”]

  • 增加键值对 （原来没有的属性名）

    • 对象名.属性名 = 值
    • 对象名[“属性名”] = 值

      数字为属性名时，只能用“对象名[“属性名”] = 值”的方式来进行添加或者获取，并且引号可以忽略不写

  • 修改 （将原有属性名的属性值进行修改）

    • 对象名.属性名 = 值
    • 对象名[“属性名”] = 值

  • 删除

    • 删除键值对 delete 对象名.属性名
    • 删除属性值 保留属性名 对象名.属性名=null

自定义属性

对象中有属性，属性分为内置属性和自定义属性

对象创建后就自带的属性，我们成为内置属性

有时候内置属性不能满足我们的需求，我们就可以在这个对象上人为添加一些属性，我们称之为自定义属性

对象的属性和变量很像，都是用来储存值的容器

买了一件衣服，衣服有两个兜子，两个兜子不够用了，那么我可以在这个衣服上继续添加其他的兜子
