#动画 animation
一些 CSS 属性是可以有动画效果的，这意味着它们可以用于动画和过渡。

动画属性可以逐渐地从一个值变化到另一个值，比如尺寸大小、数量、百分比和颜色。

因为是css3新出现的属性，所以并不是所有的浏览器都支持。

-webkit-, -moz-, 或 -o- 指定了第一个支持该属性的浏览器版本前缀。

那么如何来实现动画效果呢？
如果你熟悉动漫，就会看过一本小书，每一页画上一个动作，快速翻页时，小书上的人物或者景象就会动起来，我么将每一页都称之为帧，也叫做帧动画，那么这个也是一样滴，我么利用关键帧来实现动画效果。

以百分比来规定改变发生的时间，或者通过关键词 “from” 和 “to”，等价于 0% 和 100%。

百分比是指动画完成一遍的时间长度的的百分比 ，0% 是动画的开始时间，50%是动画完成一半的时间，100% 动画的结束时间。百分比后面的花括号写：在动画执行过程中的某时间点要完成的变化。

那么下面我们就来说说animation（动画这个属性）
animation是css3的一个新增属性，所以支持的浏览器有以下几种：

所以在使用的时候，去要添加前缀， -webkit-, -ms- 或 -moz-。

这个属性是一个复杂属性，所以有很多的分支属性，下面我们就一点一点看一下。

animation-name

指定要绑定到选择器的关键帧的名称

这个属性有两个值，第一个是绑定到选择器的关键帧名称，另一个是none，指没有设置过动画，或者覆盖掉之前的动画。

##animation-duration
[duˈreɪʃn] 持续
属性定义动画完成一个周期需要多少秒或毫秒

##animation-timing-function
指定动画将如何完成一个周期

值 描述
linear 匀速
ease[iz] 默认，低速开始，然后加快，结束前变慢
ease-in 动画以低速开始
ease-out 动画以低速结束
ease-in-out 动画以低速开始和结束
cubic-bezier(n,n,n,n) 在 cubic-bezier 函数中自己的值。可能的值是从 0 到 1 的

animation-delay

[dɪˈle] 延迟推迟

属性定义动画什么时候开始
值单位可以是秒（s）或毫秒（ms）
允许负值，-5s 使动画马上开始，但跳过 5秒进入动画。

animation-iteration-count

iteration [ˌɪtəˈreʃən] 反复 重复
count [kaʊnt] 计数 次数

定义动画的播放次数
除了可以定义次数，还可以定义为infinite，持续播放。

animation-direction

direction [dɪˈrɛkʃən] 方向
属性定义是否循环交替反向播放动画
注意：如果动画被设置为只播放一次，该属性将不起作用。

值 描述
normal 正常播放
reverse[rɪˈvɜ:rs] 反方向播放
alternate[ˈɔ:ltərnət] 奇数正向播放，偶数反向播放
alternate-reverse 奇数反向播放，偶数正向播放
initial 默认值
inherit 继承父级

animation-fill-mode

fill [fɪl] （使）充满
mode [moʊd] 方式; 状况; 时尚，风尚; 调式;

规定当动画不播放时（当动画完成时，或当动画有一个延迟未开始播放时），要应用到元素的样式。

默认情况下，CSS 动画在第一个关键帧播放完之前不会影响元素，在最后一个关键帧完成后停止影响元素。animation-fill-mode 属性可重写该行为。 　

值 描述
none 默认值 什么都不发生
forwards[ˈfɔrwədz] 在动画结束后（由 animation-iteration-count 决定）[ˌɪtəˈreʃən]动画将应用该属性值。
backwards[ˈbækwərdz] 动画将应用在 animation-delay 定义期间启动动画的第一次迭代的关键帧中定义的属性值。这些都是 from 关键帧中的值（当 animation-direction 为 “normal” 或 “alternate” 时）或 to 关键帧中的值（当 animation-direction 为 “reverse” 或 “alternate-reverse” 时）。
both 动画遵循 forwards 和 backwards 的规则。也就是说，动画会在两个方向上扩展动画属性。
initial 设置该属性为它的默认值
inherit 从父元素继承该属性

animation-play-state

指定动画是否正在运行或已暂停
| 值 | 描述 |
| :——– | ——–|
| paused[pɔzd]| 指定暂停动画 |
|running[ˈrʌnɪŋ]|指定正在运行的动画|

总结一下 ：
第一个值是动画名
第二个值是持续时间
第三个值是运动方式
第四个值是延迟时间
第五个值是执行次数
第六个值是循环方式（可不写）
第七个值是动画最后的状态（可不写）

布局
1 固定布局 px
2 百分比布局 100%（一个页面的局部）
3 rem 通过计算根元素的字体大小与设计稿还有视口宽度（只在移动端使用）
4 媒体查询（pc - pad - 移动端）
5 弹性盒模型布局 （B课程 + 3D）

一般设计师给我们的移动端设计稿的大小是640px
移动端设备屏幕宽度
iphone 5 – 》 320px
fontSize=屏幕宽度/设定基数100
fontSize=320px/640100=50px
iphone 6 – 》 375px
fontSize=375/640100=58.59375px
iphone 6p –》 414px
fontSize=414/640100=64.6875px

var winW = document.documentElement.clientWidth, desW = 640, htmlFont = winW / desW * 100;

给根元素（html）设置 font-size:100px;

rem 布局 对于字体的大小控制不是特别的完美，所以有时候页面中字体部分我们可能还会通过px来进行控制（媒体查询配合）

HTML5常用标签

HTML5 是下一代 HTML 标准。
HTML , HTML 4.01的上一个版本诞生于 1999 年。自从那以后，Web 世界已经经历了巨变。
HTML5 仍处于完善之中。然而，大部分现代浏览器已经具备了某些 HTML5 支持。

HTML5和CSS3 是两种不同的文本语言

新特性

  • 用于绘画的 canvas 元素
  • 用于媒介回放的 video 和 audio 元素
  • html4 用来播放视频使用swf （flash）
  • 对本地离线存储的更好的支持
  • 新的特殊内容元素，比如 article、footer、header、nav、section
  • 新的表单控件，比如 calendar、date、time、email、url、search
  • 更好的服务开发，更加的利于SEO优化

新增的语意标签

article 文章 文章页
aside 侧边栏
datalist 规定input的各种情况

footer 规定body內或文章內的页脚内容
header 规定body內或文章內的页头部分

details 标签规定了用户可见的或者隐藏的需求的补充细节。
