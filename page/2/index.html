函数的作用

  • js是从上到下执行代码，有的时候需要定义一个方法只有特定的条件才会执行这个方法，那么此时我们就需要这些代码，默认时不执行，只有触发是才执行

定义函数：
关键字 函数名(){要执行的js代码块};
执行函数：
函数名();

函数名;
函数名();
的区别是什么？*
函数名后面没有括号，这个函数名就是一个代表整个函数的变量，用来代表函数和储存函数；
函数名后面有括号，代表这个函数已经执行完成的结果。

函数名() 代表函数执行完成，接收到的是返回值，此处的返回值如果没有设定，接收值为 undefined，如果设定，设定的是什么接受的就是什么。

return 后面的值是什么，返回的就是什么，如果只写return不写值，返回的依然是undefined。

函数内return后面的代码不会再执行了。

return 返回值是一个值。

私有变量：函数内定义的变量，函数外不能够获取到
如果想让外界获取私有变量代表的值，需要将这个值返回到函数外，return只能讲变量代表的值返回到函数外，变量依然是私有的，谁来接收这个值呢？函数名()来接收。

函数的参数

  • 形参：就是用来接收实参的变量，用来代表和储存实参
  • 实参：传入到函数内具体的值

形参和实参是一一对应的关系，如果形参定义，而实参没有传，那么形参的返回值为undefined

形参相当于只是声明了变量，但没有赋值，传入实参后，形参被赋值。

arguments

是用来接收实参的一个对象
接收到的实参放在一个中括号中，每一个实参用逗号分隔开，我们将这个对象接收到的值叫做类数组（不是官方叫法）

函数用来接收实参的方法：

  • 形参
  • arguments

了解js

  • javascript
  • js是运行在浏览器（运行环境）
  • js是唯一一门可以和用户直接交互的语言
  • js是一门轻量级的解释型脚本语言
    • 解释型：代码边解释边运行
    • 轻量级：语言不严谨
  • js安全性低，很容易被篡改
  • js是单线程语言（从上到下执行）

js的组成

  • ECMAScript ，ECMA是一个组成，这个组织规定了js的基本规范 – ECMAScript 实际上就是js的API（js的规范）
  • DOM Document Object Model 文档对象模型 是操作document（文档对象） 文档对象是文件的顶级对象
  • BOM 浏览器对象模型(Browser Object Model) 操作的是浏览器 ，浏览器的顶级对象是window（浏览器对象）

js的引入方式

js是不能独立运行的，需要依靠html文档运行在浏览器中，所以我们需要将js引入到html中

  • 行内式
  • 内嵌式
  • 外联式

js代码一般写在整个html文档的最后面，因为要先加载document的所有内容，然后再将js添加到每一个DOM对象上

如果将js代码写在外联式的元素中间，这部分代码是无效的

parseFloat() 浮点数（小数）
parseInt() 整数

以上两个方法，都是从指定值中去分离一部分数字，一般指定的值要么是数字要么是字符串,如果传入的值不是数字也不是字符串，先将值转为字符串，然后在字符串中取数字

工作机制：将值从第一个查找到最后一个，将有效数字部分返回，Float可以识别小数，Int只识别整数部分

返回值：要么是数字 要么是NaN

Number(value) 将其他数据类型转为数字数据类型

parseFloat(value) 浮点数
parseInt(value) 整数
返回值：数字 NaN

true value不是数字
false value是数字

其他数据类型如果遇到字符串拼接的时候，会默认先转为字符串数据类型，然后在拼接
例如 []+’abc’ – > [].toString()+’abc’ – >’’+”abc” –>”abc”
{}+”0” –> {}.toString()+”0” –> “[object object]” + “0”–>”[object object]0”

Element:active 用户行为选择器 被激活，常用链接与按钮
Element:hover 用户行为选择器 鼠标停留
Element:focus 用户行为选择器 获取焦点

目标伪类选择器

选择器 描述
E:target 选择匹配E的所有元素，且匹配元素被相关URL指向

属性选择器

选择器 描述
[attribute] 选择带有相同标签属性所有元素
[attribute=value] 选择相同标签属性和属性值的所有元素
[attribute~=value] 选择属性包含指定名字的所有元素
[attribute =value] 选择指定属性名的属性值以指定字母开头的所有元素
[attribute^=value] 选择其 src 属性值以 “https” 开头的每个 <a> 元素
[attribute$=value] 选择其 src 属性以 “.pdf” 结尾的所有 <a> 元素
[attribute*=value] 选择其 src 属性中包含 “abc” 子串的每个 <a>元素

结构伪类选择器

选择器 描述
:first-of-type 选择属于其父元素的首个 <p> 元素的每个 <p> 元素
:last-of-type 选择属于其父元素的最后 <p> 元素的每个 <p> 元素
:only-of-type 选择属于其父元素唯一的 <p> 元素的每个 <p> 元素
:only-child 选择属于其父元素的唯一子元素的每个 <p>元素
nth-child(n) 选择属于其父元素的第n个指定子元素，Odd 和 even 是可用于匹配下标是奇数或偶数的子元素的关键词，使用公式 (an + b)
:nth-last-child(n) 同上，从最后一个元素计数
:nth-of-type(n) 选择属于其父元素第二个 <p> 元素的每个 <p> 元素,也可使用Odd和even与公式（an+b）
:nth-last-of-type(n) 同上，从最后一个元素计数
:last-child 选择属于其父元素最后一个子元素
:root 选择跟元素（html）
:empty 选择没有子元素的元素
:enabled 已经使用的input元素
:disabled 选择禁用的input元素
:checked 被选中的input元素
:not(selector) 选择非p元素
::selection 选择用户选择部分
